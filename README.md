## Features Demonstrated

-  **Relative links** between pages in the same project
-  **External links** to outside websites
-  **Anchor links** (jump to top / sections)
-  **New-window links** using `target="_blank"`
-  **Hover & visited** link color changes via CSS
-  **Tooltips** on hover using the `title` attribute
-  **`mailto:` links** with a pre-filled subject
